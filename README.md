# Amazon Clone

An Amazon-inspired e-commerce webpage with responsive design, including features like navigation, product sections, and a footer. This project uses HTML and CSS for structure and styling.

## Features

- Responsive navbar with logo, search bar, and user account options.
- Hero section with a promotional message and image.
- Shop section showcasing various product categories.
- Interactive footer with useful links.
